# 视频处理工具

这是一个用于批量处理视频文件的工具集，主要功能是将各种格式的视频文件转换为30fps的MP4格式，并支持视频裁剪功能。

## 功能特性

- 🎥 **多格式支持**: 支持 mp4, webm, avi, mkv, mov, flv, wmv 等常见视频格式
- ⚡ **帧率标准化**: 将所有视频统一转换为30fps
- ✂️ **智能裁剪**: 支持指定时间区间进行视频裁剪
- 🎨 **高质量输出**: 使用H.264编码和AAC音频，确保最佳质量
- 📊 **详细信息**: 显示视频的详细信息和转换参数
- 🎯 **自动命名**: 根据转换参数自动生成输出文件名

## 脚本说明

### webm2mp4.sh

主要的视频转换脚本，具有以下功能：

#### 基本用法

```bash
./webm2mp4.sh [视频路径] [入点秒数] [出点秒数]
```

#### 参数说明

- **视频路径**: 要转换的视频文件路径（必需）
- **入点秒数**: 开始裁剪的时间点（可选）
- **出点秒数**: 结束裁剪的时间点（可选）

#### 使用示例

```bash
# 完整转换（不裁剪）
./webm2mp4.sh "video.mp4"

# 从第5秒裁剪到第15秒
./webm2mp4.sh "video.webm" 5 15

# 从开始裁剪到第10秒
./webm2mp4.sh "video.avi" 0 10
```

#### 转换设置

- **目标帧率**: 30 fps
- **视频编码**: H.264 (libx264)
- **音频编码**: AAC (128k)
- **质量设置**: CRF 23 (中等质量)
- **编码预设**: medium

#### 输出文件命名规则

- 完整转换: `原文件名_30fps.mp4`
- 裁剪转换: `原文件名_开始时间s-结束时间s_30fps.mp4`

## 系统要求

### 必需软件

- **ffmpeg**: 用于视频转换
- **ffprobe**: 用于视频信息分析
- **jq**: 用于JSON解析
- **bc**: 用于数学计算

### 安装依赖

#### Ubuntu/Debian
```bash
sudo apt update
sudo apt install ffmpeg jq bc
```

#### CentOS/RHEL
```bash
sudo yum install ffmpeg jq bc
# 或者使用 dnf
sudo dnf install ffmpeg jq bc
```

#### macOS
```bash
brew install ffmpeg jq bc
```

## 使用步骤

1. **准备视频文件**: 将要转换的视频文件放在脚本同目录下
2. **设置执行权限**: `chmod +x webm2mp4.sh`
3. **运行脚本**: 根据需要使用相应的命令
4. **确认转换**: 脚本会显示视频信息并询问是否开始转换
5. **等待完成**: 转换完成后会在同目录生成输出文件

## 功能特点

### 智能处理
- 自动检测视频格式和编码信息
- 自动调整分辨率为偶数（避免编码问题）
- 智能处理不同帧率格式

### 用户友好
- 彩色输出，信息清晰易读
- 详细的视频信息显示
- 转换前确认机制
- 完整的错误处理

### 质量保证
- 使用行业标准的H.264编码
- 合理的CRF值确保质量与文件大小的平衡
- 保持原始分辨率（仅调整奇数为偶数）

## 注意事项

1. 确保有足够的磁盘空间存储输出文件
2. 转换时间取决于视频长度和系统性能
3. 建议在转换前备份原始文件
4. 某些特殊编码的视频可能需要额外的时间处理

## 故障排除

### 常见问题

1. **权限错误**: 确保脚本有执行权限 `chmod +x webm2mp4.sh`
2. **文件不存在**: 检查视频文件路径是否正确
3. **依赖缺失**: 确保已安装ffmpeg、jq、bc等依赖
4. **格式不支持**: 脚本会尝试转换不支持的格式，但可能失败
